# Static assets

Drop these files here (they are served from the site root):

- **`cv.pdf`** — your downloadable CV. Linked from the hero ("Download CV") and the
  nav. Referenced by `profile.cvPath` in `src/data/profile.ts`.
- **`og-image.png`** — social preview image (1200×630). Shown when the site is shared
  on LinkedIn / X / WhatsApp. Until you add it, the meta tag points to a missing file
  (harmless, just no preview thumbnail).

`favicon.svg` is already here.
